## Prerequisites

### Installing Git

#### Windows

Go to [this Link](https://git-scm.com/download/win) and click either the 32-bit standalone or the 64-bit standalone based on your device

#### Mac OS

Open the Terminal application and type `git --version` into it, a popup should appear guiding you through the installation, if it didnt appear you already have git installed!

#### Linux

Open the terminal and use your package manager to install git

### Installing VSCode

To install VSCode go to [this link](https://code.visualstudio.com/) and press the download button
